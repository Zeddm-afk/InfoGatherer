# aiqicha
简介:输入企业名获取企业介绍及账号相关信息

用途:

- 收集企业信息便于提交漏洞[企业名，业务，行业，创始人，所在地]
- 收集企业账号信息用于渗透参数[手机号，邮箱]
- 程序执行后会将相关文件保存至当前目录下

用法:

```
python 企业账号信息收集.py 
```
